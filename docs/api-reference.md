# 📚 API Reference

## Account APIs

| Method | Endpoint           | Description                  |
| ------ | ------------------ | ---------------------------- |
| POST   | /api/accounts      | Create Account               |
| GET    | /api/accounts      | Get All Accounts _(Planned)_ |
| GET    | /api/accounts/{id} | Get Account _(Planned)_      |
| PUT    | /api/accounts/{id} | Update Account _(Planned)_   |
| DELETE | /api/accounts/{id} | Delete Account _(Planned)_   |

## Response Codes

| Code | Meaning               |
| ---- | --------------------- |
| 200  | Success               |
| 201  | Created               |
| 400  | Validation Error      |
| 404  | Resource Not Found    |
| 409  | Duplicate Resource    |
| 500  | Internal Server Error |
